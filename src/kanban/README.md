# Awesome Kanban Board

## Необходимое ПО
 - Google Chrome
 - VS Code
 
 ## Начало работы
 1. Для запуска потребуется установить расширение VS Code - Live Server(https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).
 2. Если ваш браузер по умолчанию не Chrome,то потребуется в настройках расширения Live Server поменять на Chrome(https://fooobar.com/questions/15327406/how-to-change-default-browser-with-vs-codes-open-with-live-server).
 3. Затем необходимо скачать файлы представленные файлы(index.html и всю папку src).
 4. Перейти в VS Code и выбрать File - Open Folder - Выбрать папку, в которую помещены скачанные файлы.
 5. В появившемся списке слева(Если он скрыт, то нужно кликнуть на пункт панели слева Explorer) щёлкнуть по файлу index.html. Файл откроется для редактирования.
 6. Правой кнопкой мыши вызвать контекстное меню из окна редактирования и выбрать в нём "Open With a Live Server". Или щёлкнуть по надписи "Go Live" в голубой строке состояния, расположенной внизу редактора.
 7. Откроется браузер Google Chrome(Если он указан в настройках расширения Live Server) с Awesome Kanban Board.
 
 Подобные расширения других редакторов, позволяющие запускать локальные сервера, возможно, также корректно смогут развернуть эту страницу.
 
 ## Структура
 В файле index.html хранится html-код.
 В папке src хранятся три папки: fonts, styles, scripts.
 - fonts хранит в себе шрифты страницы.
 - styles хранит все стили страницы, которые объединяются в главном стиле index.css.
 - scripts хранит в себе все скрипты, разбитые по папкам разделов страницы, изменения в которых они производят.
